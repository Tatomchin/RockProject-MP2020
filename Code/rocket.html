Introduction

The Magnavox Odyssey released by Magnavox in the United States in September 1972 connects to a television set, and two rectangular controllers attached by wires. The Odyssey lacks sound capability and can only display monochrome white shapes on a blank black screen.

Plastic overlays

The games include plastic overlays which stick to the television via static cling, to create visuals for the game. Different games that use the same game card can have different overlays.
